# Running the project

You can see a live demo of this project [here](https://devsys-app.azurewebsites.net/account/login?returnUrl=%2F).

You can also run this app locally.  To run this project locally you will need to have installed:

1. .Net SDK v9
2. Node.js (v20 or later)
3. SQL Server (LocalDB or Express)
4. Angular CLI (npm install -g @angular/cli)

```bash
# you will need git installed to run this
git clone https://github.com/andrea-muresan/device-management-system.git
cd device-management-system
```

5. Open SQL Server Management Studio (SSMS) and run the next scripts from DatabaseScripts in the following order:
- CreateDB.sql
- CreateTables.sql
- PopulateTables.sql
- AlterUsersTables.sql

6. Restore the packages by running:

```bash
# From the solution folder (device-management-system)
dotnet restore

# Change directory to client to run the npm install.  Only necessary if you want to run
# the angular app in development mode using the angular dev server
cd client
npm install
```

7. To use the generating description for devices you will need an ai studio key. You can get one [here](https://aistudio.google.com/api-keys?project=gen-lang-client-0740262472)
```bash
# run this from the API folder
dotnet user-secrets init
dotnet user-secrets set "GoogleAI:ApiKey" "ADD_YOUR_KEY"
```

8. You can then run the app and browse to it locally by running:

```bash
# run this from the API folder
cd API
dotnet run
```

9. You can then browse to the app on https://localhost:5001
10. If you wish to run the Angular app in development mode you will need to install a self signed SSL certificate.  The client app is using an SSL certificate generated by mkcert. To allow this to work on your computer you will need to first install mkcert using the instructions provided in its repo [here](https://github.com/FiloSottile/mkcert), then run the following command:

```bash
# cd into the client ssl folder
cd client/ssl
mkcert localhost
```

11. You can then run both the .Net app and the client app.

```bash
# terminal tab 1
cd API
dotnet run

# terminal tab 2
ng serve
```

12. Then browse to [https://localhost:4200](https://localhost:4200)
